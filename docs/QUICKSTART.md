# Quick Start Guide

## Get Started in 3 Steps

### 1. Install
```bash
git clone https://github.com/Kartikeya-trivedi/ai_interviewer.git
cd ai_interviewer
pip install -r requirements.txt
```

### 2. Run Demo
```bash
python -m src.main --demo
```

### 3. Explore Features
```bash
# Check platform support
python -m src.main --info

# Get help
python -m src.main --help
```

## What You'll See

The demo mode will:
- Display your platform information (Windows/Linux)
- Confirm platform compatibility
- Show sample interview questions
- Demonstrate the core functionality

## Next Steps

1. Explore the configuration in `config.yaml`
2. Check out `docs/DOCUMENTATION.md` for detailed API reference
3. Run tests: `python -m unittest tests.test_interviewer -v`
4. Start building your custom interview scenarios!
